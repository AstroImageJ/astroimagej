Plot Window Help

Title, axes, and legend control is done through Multi-plot Main.

Keybindings Map

Action Result
Measurements
Right-click drag Measure distance
Middle-click drag Measure distance and save to log
Middle-click Log plot coordinates
Zoom Controls
Scroll Zoom
Scroll+Alt/Opt Zoom in y only
Scroll+Ctrl/Cmd Zoom in x only
Drag+Alt/Opt Define a region to zoom plot to
Right-click Reset plot zoom
Navigation
Drag Pan plot
Table Interaction
Shift and hover Highlight datum and select row in table
Shift+left-click Remove data point from plot and table
Shift+right-click Restore data point to plot and table
Marker Control
Ctrl/Cmd+left-click Set left marker position
Ctrl/Cmd+right-click Set right marker position
Ctrl/Cmd+Alt/Opt+Left-click Set left marker position and adjust right marker to keep separation the same as transit markers
Ctrl/Cmd+Shift+Left-click Set left trim marker
Ctrl/Cmd+Shift+Right-click Set right trim marker
Miscellaneous
Ctrl/Cmd+H Open this help panel
Ctrl/Cmd+X Copy x-coordinate of the cursor
Ctrl/Cmd+Y Copy y-coordinate of the cursor
Ctrl/Cmd+C Copy plot
