### Includes
ImageJ 1.54m1 and Java 25

## General
  - Adds various error dialogs for updater failure modes.

## New Features
  - Adds a feature that optionally allows border regions to be defined around the edges of the detector. The regions can be used to 
mask areas that are not to be used when plate-solving and multi-aperture auto comparison star selection.
  - Adds a feature that finds the common field of view for all images in a stack of images with WCS solutions (i.e. are plate-solved). 
This feature is currently used to limit the image region used by auto comp star selection to avoid selecting reference stars
that are not in all images. This option can be set to run automatically when a plate-solved stack is opened.
  - The two new features above can be accessed in the new "Display region exclusion" button above the image stack, as well as in
The Multi-Aperture and Astrometry set-up panels. Excluded regions will be shaded red in the image display. 
Right-click the new icon to access the settings from the stack display.

## Bug Fixes
  - Macro recording is now fixed in the Multi-Aperture set-up panel.
