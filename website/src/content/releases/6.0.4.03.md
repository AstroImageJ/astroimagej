### Includes
ImageJ 1.54m1 and Java 25

## New Features
  - Enable read measurementstable tool on Toolbar to read fits.fz measurements tables.
  - Extend matched measurements table extensions to include csv and tsv file suffixes.

## Bug Fixes
  - Fixed multi-aperture macro recording default file name.
  - Fixed macro reading of apertures file.
  - Fixed TESScut incorrect headers when skipping images due to TESS quality flag.
  - Fix incorrect handling of multiple file extensions.
