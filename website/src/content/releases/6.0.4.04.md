### Includes
ImageJ 1.54m1 and Java 25

## New Features
  - ImageJ using interface scaling mostly supported using option at AIJ_Toolbar > Edit > Options > Appearance > Gui Scale (plot scaling is still a work in progress).
  - Add envrionment variable for AIJ run task to copy prefs file from a known location.
  - Support launching AIJ and directly opening an image sequence.

## Bug Fixes
  - Fixed WCS problem when opening fits files.
  - Fixed import image sequence problem.
  - Fixed NPE when toggling aperture names with no apertures.
