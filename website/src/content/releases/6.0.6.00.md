### Includes
ImageJ 1.54s and Java 25

## General
  - Multiplot - change sense of odd/even transit notation to match SPOC / QLP
  - Periodogram - changed sense of odd/even transit notation to match SPOC / QLP

## Bug Fixes
  - FITS reader: fixed reading of TESScut images.
  - FITS reader: fixed LCO image detection.
  - Stack Display: reverted stack window changes from previous build (which caused Mac stack display issues).
  - Stack Display: prevent subtitle from being hidden on Wayland-based Linux systems.
  - AIJ Toolbar: fixed AIJ icon in toolbar and in linux taskbar.
  - General: Fixed window centering on multi-monitor Wayland systems (requires wayland-info utility to be installed on linux system).
 
### Other
  - Set zgc as default java garbage collector on supported systems (helps reduce the amount of memory used when opening large stacks).
