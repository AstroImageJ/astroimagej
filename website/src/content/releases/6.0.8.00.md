### Includes
ImageJ 1.54s and Java 25

## General
  - Builds are now provided for Linux aarch64 (ARM64) platforms, 
    including Raspberry PI platforms. 

## New Features
  - Image Scaling (for high DPI monitors) is now supported at 
    `AIJ_Toolbar > Edit > Options > Appearance > GUI Scale`. 
    This scaling method has been optimized
    for all AIJ windows and plots are drawn as vectorized images 
    so will remain sharp and unpixelated even for higher scaling factors.

## Bug Fixes
  - AIJ measurements table files that are saved with the .xls filetype, 
    but are really tab-delimited (tsv) files, can now be reopened into AIJ again. 
    Files with filetype .xls, that are actually Excel formatted files, will still
    be rejected since AIJ cannot directly read Excel formatted files.
  - A bug that cause plot panning to sometimes resize the plot has been fixed.
  - Double-clicking a fits file within a zip file will now open in AIJ (if
    the file association has been set up in the OS).
