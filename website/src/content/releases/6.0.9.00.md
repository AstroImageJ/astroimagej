### Includes
ImageJ 1.54s and Java 25

## General
  - Aperture overlays and cursor are now drawn more smoothly, fixing offset between where aperture center 
crosshair and circle are displayed. See [issue 77](https://github.com/AstroImageJ/astroimagej/discussions/77).
  - Change BJD_UTC interface from (now defunct) OSU website to Shared Skies website.

## New Features
  - Automatically scale ImageJ plots (as opposed to multi-plot), in most cases.
  - Add the "Subtract Column" operation as a mathematical operator in measurements table panel (right click on column).

## Bug Fixes
  - Fix lockup on macOS when opening images when main window is on a second monitor.
  - Fix rare cases when the plot would not update.
  - Fixed various Hi-DPI scaling bugs.
  - If updating from IJ 6.0.8.00, remove message indicating that there are two 3D plot plugin files.
  - Fix Multi-plot bug that resulted in unscaled plot dimensions when GUI scale was set to 1.1.
 
### Other
  - Linux update script now deletes files in old AIJ installation before writing new files.
  - Added more shape support for scaled imagej-style plots.
  - Restore normal IJ plot window buttons for normal IJ plots
